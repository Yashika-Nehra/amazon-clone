# Amazon Clone - Web Development Project

## About the Project

This is a front-end web development project where I have created a clone of the Amazon India (amazon.in) website using React.js. The goal of this project was to learn and practice core web development concepts like component-based design, routing, state management, and responsive layouts.

This project was built as part of my B.Tech curriculum to understand how real-world e-commerce websites are structured and developed.

---

## What I Learned

- How to set up and work with a React application
- Breaking a webpage into reusable components
- Managing state (like cart items) across multiple pages
- Using React Router to navigate between pages without reloading
- Writing responsive CSS that works on mobile and desktop
- Using Git and GitHub to manage and upload code

---

## Features

- Home page with a banner, category section, and product listings
- Search bar that filters products in real time
- Filter products by category (Electronics, Books, Fashion, etc.)
- Sort products by price, rating, or discount
- Product detail page with description and buy options
- Add to cart and remove from cart functionality
- Cart page showing items and total price
- Works on mobile, tablet, and desktop screens

## How to Run This Project

Make sure you have Node.js installed on your computer. You can download it from https://nodejs.org

Step 1 - Clone the repository:
```
git clone https://github.com/Yashika-Nehra/amazon-clone.git
```

Step 2 - Go into the project folder:
```
cd amazon-clone
```

Step 3 - Install the required packages:
```
npm install
```

Step 4 - Start the project:
```
npm start
```

The website will open in your browser at http://localhost:3000

---

## Project Structure

```
src/
  components/       - Reusable parts like Navbar, Footer, ProductCard
  pages/            - Full pages like Home, Cart, Product Detail
  data/             - Product and category information
  assets/images/    - Images used in the project
  App.js            - Main file that connects all pages
  index.js          - Starting point of the React app
```
